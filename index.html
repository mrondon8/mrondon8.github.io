MaterialTracker

The Material e-Tracker is a tool to allow users to keep track of materials from one place to another. Thousands of materials are moved between warehouses, sales-units, and outlets. Certain materials required approval from a warehouse manager in order to be moved with a built in email alert system that handled the approval notification process. The system enabled users to track up-to-date location on the materials.

PatientDoctor

This system allows patients that are registered to doctors to schedule an appointment with them, along with providing them with a plethora of information. This tool also allows the doctors to see the requested appointment and whether to approve it or not. If approved the tool sends an e-mail to the patient about their confirmed appointment.

TestGenerator

A test generating software that incorporated technologies such as Struts, Validator, Tiles, Java, Oracle Database and Javascript/Jquery. The user would login, then pick which categories he would like to be tested on and how many questions the test should be. After submitting the test would be graded and the user would be shown a report of correct and incorrect answers.
